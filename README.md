# [GameOffal](https://develoder.github.io/GameOffal/)

Одностраничное приложение для личных обзоров игр. Карточки с оценками по параметрам, радар-диаграммы, тиры и экспорт в PDF. Все данные хранятся в одном JSON-файле — без сервера и базы данных.

## Возможности

- **Карточки игр** — название, обложка, оценки по 7 параметрам (сюжет, геймплей, визуал, тех. состояние, реиграбельность, сложность, вайб), радар-диаграмма, средняя и личная оценка
- **Тиры S–F** — присвоение тира каждой игре, порядок внутри тира, drag & drop для сортировки
- **Два режима просмотра** — «Обзор» (карточки) и «Тиры» (таблица с перетаскиванием игр между рядами)
- **Экспорт в PDF** — выгрузка текущего режима (обзор или тиры)
- **Тёмная тема** — переключатель в шапке, выбор сохраняется в localStorage
- **Палитра из обложки** — рамка карточки подстраивается под доминантный цвет изображения
- **Drag & drop** — загрузка JSON на страницу, загрузка изображений при редактировании карточки
- **Тултип в режиме «Тиры»** — при наведении на игру показывается радар и описание

## Запуск

Откройте `index.html` в браузере. Загрузите JSON через меню или перетащите файл на страницу, либо нажмите «Создать обзор» для пустого проекта.

В репозитории есть файл `games.json` — пример данных для тестирования приложения.

## Использование

1. **Загрузка данных** — нажмите кнопку создать обзор или перетащите его на страницу
2. **Редактирование** — на карточке: карандаш — редактировать, корзина — удалить; кнопка «+» добавляет новую карточку
3. **Название темы** — нажмите на карандаш рядом с заголовком
4. **Сохранение** — кнопка «Сохранить как...» сохраняет JSON (для диалога выбора папки включите в браузере «Спрашивать, куда сохранять файлы»)
5. **Режим «Тиры»** — перетаскивайте игры между рядами или внутри ряда для изменения тира и порядка

## Изображения

Изображения хранятся **в самом JSON-файле** (base64). При перетаскивании или выборе картинки в форме редактирования она встраивается в данные карточки. Отдельный сервер для картинок не нужен.

## Технологии
